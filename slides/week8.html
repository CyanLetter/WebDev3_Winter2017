Flexbox

What is Flexbox?

Layout specification which aims to make component layouts more adaptive to dynamic sizes.

Use it when you need it

You can use flexbox with floats and other features.

Some things are more convenient with flexbox.

Helpful to limit use to individual components.

Background

Hierarchy

Properties apply to both parent (container) and direct child (item) elements

Basics

Define a flex container


.parent {
  display: flex;

  .child {
    /* Layout is controlled by parent */
  }
}

Parent Properties

flex-direction

Flexbox is a single direction layout.


.parent {
  flex-direction: column;
  /* row (default), row-reverse, column-reverse */
}

justify-content

Where to add empty space.


.parent {
  justify-content: flex-start; /* default */
  /* center, flex-end, space-around, space-between */
}

align-items

How to align items of differing widths or heights.

Sort of like vertical-align with inline-block.


.parent {
  align-items: flex-start; /* default */
  /* center, flex-end, baseline, stretch */
}

More below... but we will come back to those

flex-wrap

Whether content wraps to multiple lines


.parent {
  flex-wrap: nowrap; /* default */
  /* wrap, wrap-reverse */
}

align-content

How to align multiple lines of flex content.


.parent {
  align-content: stretch; /* default */
  /* flex-start, center, flex-end, space-around, space-between */
}

Child Properties

order

Modify the order of individual items.

The z-index of flexbox.


.child {
  order: 99;
}
.child-2 {
  order: -1;
}

flex-grow

Allows a flex element to expand.

Proportionally, how much extra space should this element fill.


.child {
  flex-grow: 1;
}
/* default 0 */

flex-shrink

Lets an element shrink.

Works best paired with flex-grow.


.child {
  flex-shrink: 1;
}
/* default 1 */

flex-basis

The baseline size of a flex element.

Uses width or height by default.


.child {
  flex-basis: 20%;
}
/* default auto */
