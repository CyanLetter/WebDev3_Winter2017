Not Everything is a jQuery Plugin

'jQuery Plugin' is often misused.

Many things can be done without jQuery.

Maybe you don't need jQuery.

Be aware of whether your library uses jQuery.

ScrollTo

Github Repo

This requires jQuery.

Setup

Remember to include the plugin AFTER jQuery

Simple Usage


$("#scrollButton").click(function(){
	
	// the first parameter is the target, 
	// the second is the duration.
	$.scrollTo( $("#scrollTarget"), 400 );

});

Other Targets


// absolute number
$.scrollTo( 150, 400 );

// relative number (incrementing)
$.scrollTo( '+=100px', 400 );

// absolute number (object)
$.scrollTo( { top: '200px', left: '50px'}, 400 );
